## Aplicación React con Autenticación
Tecnologías usadas:
```
- React
- Typescript
- TailwindCSS
- Zustand
- Toastify
- Axios

- Express
- MongoDB
- Bcryptjs
- Cookie-parser
- Cors
- Jsonwebtoken
- Nodemailer
```
